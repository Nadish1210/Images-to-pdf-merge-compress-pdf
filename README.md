# Its Nadish - Image to PDF Converter

A powerful and easy-to-use web tool to convert images to PDF, merge multiple PDFs, and compress PDFs. Built with Streamlit.

![Its Nadish](https://img.shields.io/badge/Made%20with-Streamlit-FF4B4B?style=for-the-badge&logo=streamlit)

## ✨ Features

- **🖼️ Images to PDF**
  - Convert multiple images (JPG, PNG, WEBP, etc.) into a single PDF
  - Support for A4, Letter, Legal & Original page size
  - Portrait & Landscape orientation
  - Quality control (Low, Medium, High, Maximum)
  - Proper image fitting and centering

- **📑 Merge PDFs**
  - Merge multiple PDF files into one

- **📉 Compress PDF**
  - Reduce PDF file size while maintaining good quality

- **⭐ Feedback System**
  - Users can give feedback and view all feedbacks

- **🔄 Reset Everything**
  - One-click reset for all uploaded files

- **Frontend**: Streamlit
- **Backend**: Python
- **PDF Handling**: ReportLab, PyMuPDF (fitz), PyPDF2
- **Image Processing**: Pillow (PIL)
